# Glossary (Beginner Friendly)

- **Schema**: The structure of the database (tables, columns, types).
- **Migration**: A small change to the schema (add a column, add an index).
- **Seed data**: Pre-filled sample data used for testing or demos.
- **Primary key (PK)**: Column (or columns) that uniquely identify a row.
- **Foreign key (FK)**: Column that references a primary key in another table.
- **Index**: Data structure that makes certain queries faster.
- **Domain**: A logical area of the business (auth, billing, CRM, etc.).
